API Documentation

Welcome to the CyberSafe UK API. This API provides programmatic access to threat intelligence, incident reporting, and cybersecurity statistics for the United Kingdom.

Base URL

https://api.cybersafe.uk/v1

Authentication

All API requests require an API key passed in the header:

Authorization: Bearer YOUR_API_KEY

Endpoints

GET /threats/live

Get real-time threat intelligence data

{ "status": "success", "data": { "total_attacks": 15234, "blocked": 12891, "active": 42, "top_types": ["phishing", "ransomware", "ddos"] } }
POST /incidents/report

Submit a cybercrime incident report

{ "category": "phishing", "description": "Suspicious email received...", "evidence_urls": ["https://..."], "severity": "medium" }
GET /statistics/uk

Get UK cybersecurity statistics and trends

Rate Limits

  • Free tier: 100 requests/day
  • Professional: 10,000 requests/day
  • Enterprise: Unlimited
