### ğŸ“¥ Installation

1.  **Clone the repository**
2.  **Install Backend Dependencies**:
    ```bash
    cd UK/backend
    pip install -r requirements.txt
    ```
    *(Ensure you have `fastapi`, `uvicorn`, `requests`, `beautifulsoup4`, `tinydb` installed)*

3.  **Setup Backend Data**:
    The backend uses a local JSON database (`cyber_chatbot_UK1.json`). Ensure this file is present or generated by the ingestion tools.

### ğŸƒ Running the Application (Automated)

We have provided a convenient batch script to launch everything at once.

Double-click **`run_chatbot.bat`** in the root directory.

This script will:
1.  Launch the **FastAPI Backend** in a new terminal window (listening on port `8001`).
2.  Wait for the server to initialize.
3.  Open the **UK Dashboard** (`UK/index.html`) in your default web browser.
